## Quick Start

**Windows:**
```powershell
irm https://raw.githubusercontent.com/YourName/Redacted/main/scripts/run.ps1 | iex
```

**Linux / macOS:**
```bash
curl -fsSL https://raw.githubusercontent.com/YourName/Redacted/main/scripts/run.sh | bash
```

## Requirements

- Unlocked bootloader
- OrangeFox / TWRP recovery (for full access)
- ADB / platform-tools (auto-installed if missing)
- Python 3.10+ (auto-installed if missing)
